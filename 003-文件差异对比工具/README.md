# 文件差异对比工具

一个类似 Git Diff 的文本比较工具，支持高亮差异、并排对比和合并视图。

## 功能特性

- **文本对比**: 支持输入两段文本并高亮显示差异（新增、删除、修改）
- **视图模式**: 
  - 并排对比模式：左右分别显示原始文本和新文本
  - 合并视图模式：在一个视图中合并显示所有差异
- **点击定位**: 点击差异块可以高亮并定位到对应的位置
- **差异导航**: 使用控制面板中的"上一个/下一个"按钮快速在差异之间导航
- **差异统计**: 显示新增、删除、修改、不变行数以及总变更行数（带渐变背景的醒目标识）
- **示例数据**: 提供三个预设示例，包括一个较大的文本用于测试
- **虚拟滚动**: 已实现 VirtualList 组件，支持处理大文本（可按需集成）

## 技术栈

- Vue 3 (Composition API)
- Vite
- 原生 JavaScript 实现的 Myers diff 算法

## 关键设计

### Diff 算法

实现了基于 Myers 算法的行级 diff：
- 高效计算文本差异
- 支持删除、插入、相等和修改行的识别
- 复杂度 O((M+N)D)，其中 D 为差异数量

### 修改识别

通过计算两个相邻行（一删一插）的文本相似度，将相似度超过阈值的识别为修改行：
- 相似度计算：匹配字符数 / 最大长度
- 默认阈值：30%

### 虚拟滚动

已实现 VirtualList 组件，支持处理大文本（1000+ 行），仅渲染可见区域的行，提升性能。

### 组件设计

- **输入区**: 双列布局，支持清空操作
- **控制区**: 视图切换、差异统计、差异导航
- **Diff 区**: 支持两种视图模式，点击定位差异

## 安装和运行

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

应用将在 http://localhost:3003 启动。

### 构建生产版本

```bash
npm run build
```

## 验证步骤

1. 启动应用后，点击下方三个示例按钮之一：
   - **加载示例 1**: 展示函数修改的简单对比
   - **加载示例 2**: 展示数组和统计计算的对比
   - **加载大文本**: 测试处理较多行数的文本
2. 在左右两个文本框中输入或粘贴文本，观察差异高亮
3. 点击控制面板中的"并排对比"和"合并视图"按钮切换显示模式
4. 点击控制面板中的"上一个/下一个"按钮导航差异
5. 点击任意差异行（新增、删除或修改），观察高亮和定位
6. 查看上方控制区的差异统计信息（变更行数带渐变背景高亮显示）
7. 点击"清空"按钮可以清空对应区域的文本

## 使用说明

1. 在左侧"原始文本"区域输入或粘贴原始文本
2. 在右侧"新文本"区域输入或粘贴修改后的文本
3. 差异会自动高亮显示：
   - 红色背景：被删除的行
   - 绿色背景：新增的行
   - 黄色背景：被修改的行
4. 使用控制面板切换视图模式或导航差异
5. 点击任意差异行可以高亮并定位
6. 使用"清空"按钮清空对应区域的文本
